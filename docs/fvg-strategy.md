# FVG 失衡缺口策略 — 定版規格(v2.3)

> **v2.3 更新(2026-06)** — 經 8 幣 2017–2026 全資料 walk-forward 止損/止盈掃描 + 滑價壓力測試 + 確認進場對照,改三件事:
> 1. **止損 4W→3W、止盈 5W→6W**(全表 OOS 全正且**單調**:止損越緊+止盈適中→每筆 R 越高;選 3/6 為穩健中段,不取帳面最佳的 2/5~2/8,因緊止損會使下單量加倍、實盤逆選擇更糟)。
> 2. **新增「止盈先到撤殘單」**:價格先觸及止盈位、掛單尚未全部成交 → 立即撤掉未成交限價單(否則殘單會在日後崩跌時補成交=正好接到插穿的輸單)。回測會少賺 ~0.046R,但**少的是最靠「完美成交」撐著、最不真實的那段獲利** → 更抗逆選擇、回撤更低。
> 3. **正式採雙槽多空 + 多幣組合**(見下),非單純只做多。
>
> 狀態:**參數鎖定**(3/6 是 walk-forward 穩健中段,非挑 OOS 尖點)。本文件為唯一參數來源,定版後不再調參。
> 唯一未實證者:**實盤掛單逆選擇(真實成交率)→ 需 testnet paper-trade**。

## 適用範圍
- 標的:**流動性高的主流加密籃子**——BTC / ETH / XRP / BNB / LTC / ADA / BCH / LINK(同一套參數皆 OOS 正)。SOL / ZEC 也通過。
- 時框:**1h**。
- ❌ 不可做:代幣化商品/股票(XAU/XAG/SOXL,市場微結構不同、OOS 翻負);新幣(歷史短、單一牛市)別當主力。

## 訊號:FVG(Fair Value Gap,失衡缺口)
三根 K `[g-1] [g] [g+1]`(中間大實體):
- **多頭 FVG(支撐型,做多)**:`low[g+1] > high[g-1]`,缺口區 `[high[g-1], low[g+1]]`。
- **空頭 FVG(壓力型,做空)**:`high[g+1] < low[g-1]`,缺口區 `[high[g+1], low[g-1]]`。
- 缺口寬門檻:`缺口寬 / 價格 > 0.3%`(MS=0.003)。
- 缺口於 `[g+1]` 收盤確認(因果,不偷看)。

## 進出場規則(缺口寬 `W = top − bot`)
| 項目 | 規則 |
|---|---|
| **進場** | 缺口**上緣 / 中間 / 下緣** 三點各掛 ⅓ 限價單(maker) |
| **掛單有效期(新鮮度)** | **168 根(1h = 一週)** 內未成交 → 取消該缺口 |
| **止損** | 多:`bot − 3W`;空:`top + 3W`(市價 / taker) |
| **止盈** | 多:`top + 6W`;空:`bot − 6W`(單一止盈,limit / maker) |
| **最長持有** | 200 根後仍未觸發 → 收盤平倉 |
| **每筆風險** | 1%(止損寬 → 同風險下單量小,回撤可控) |
| **同時持倉** | 每槽一次一倉;雙槽=多/空各一(前倉未平則跳過後續同向缺口) |
| **部位生命週期** | ① 止盈或止損觸發 → 立即撤所有未成交限價單、部位歸零。② **止盈位先到、掛單尚未全成交 → 撤掉殘單**(避免日後崩跌補成交=接插穿輸單) |
| **組合上限** | 全幣全槽**同時最多 15 倉**(超過則略過新訊號);8 幣約 146 筆/月 |

## 回測表現(8 幣組合, 2017–2026, 0.10% 止損滑價, 15 倉, 每筆 1%)
| 配置 | 每筆 avgR | 勝率 | 平均持有 | 月均(flat) | 最大DD |
|---|---|---|---|---|---|
| **限價 3/6 + 止盈撤殘單** | +0.161 | 39% | 1.8 天 | **+23%** | −7.6% |
| 限價 3/6(無撤單,最理想) | +0.207 | 42% | 1.6 天 | +30% | −9.4% |
| 收盤確認 3/6(成交確定) | +0.058 | 44% | 1.7 天 | +8% | −12.8% |

- **R/天(資金效率)**:緊止損版最高(1.2–1.8 天結清);止盈不必拉到 8W,**遠止盈雖墊高 avgR 但拖慢周轉**,速度校正後 6W 比 8W 划算。

## ⚠️ 真實月報酬預期:+8% ~ +23%(不是 +40%)
回測的高月報酬 = **周轉率 × 每筆 edge**,而 edge 被「完美成交」高估。用三種機制把現實夾出來:
```
完美成交上界 ………………… +30%/月  (含殘單假贏單,最不可信)
限價+止盈撤殘單 …………… +23%/月  (拿掉最可疑那塊,較可信上界)
收盤確認(成交確定,下界)… +8%/月  (確認稅可能扣太兇)
真實值落在 +8% ~ +23%/月,取決於實盤真實成交率 → 只有 paper-trade 能定。
```
- 逆選擇本質:限價單無未來資訊,**贏單常碰一下彈走→接不到;輸單插穿→一定接到**。「止盈撤殘單」與「收盤確認」是僅有的兩個對抗手段。
- 收盤確認版**成交是確定的**(市價進場無搶單問題)→ 若要先上線一個「保證可執行」的版本,用它(+8%/月仍世界級)。

## 驗證(為何相信 edge 是真的)
- **Walk-forward 止損/止盈全表掃描**:SM∈{2..6}×TM∈{3..8} **30 格全部 OOS 正**,且單調(緊止損+適中止盈→R↑)→ 是真實結構(順勢小賠大賺),非雜訊尖點。
- **滑價壓力**:止損滑價加到 0.20% 排名不變(止損距離 ≥0.6% 價格,吃不掉)。
- **逐年全正**:2017–2026 每年正,含 2018/2022 大熊市 → 非牛市漂移。
- **同幾何隨機進場對照 = −0.285R**:edge 確來自「進在 FVG」,非引擎造假。
- **崩盤抗性**:雙槽空單在崩盤月獲利(LUNA、FTX 後皆正)。
- 引擎健檢:作弊版(偷看未來)會大賺 → 證明引擎能抓正 edge。

## 已知 caveat / 風險
1. **實盤逆選擇未實證**(唯一未解)→ paper-trade 量真實成交率,才能把 +8~23% 收窄成單一數字。
2. **緊止損 = 同風險下單量加倍** → 限價單更大、更難全成交 → 別把 SM 推到 2W 以下。
3. 樣本期偏多頭;雖 2022 熊市仍正,結構性長空頭/長盤整行為可能不同。
4. 參數已鎖,**禁止再微調**(每多挑一次 OOS 最佳 = 過擬合 OOS)。
